# Photo Booth

Simple browser-based photobooth app with camera capture, filters, and save/restart controls.

## Project structure

- `photo/index.html` — UI and camera layout
- `photo/script.js` — camera, countdown, filter, and photo capture logic
- `photo/style.css` — styling for the photobooth interface

## Usage

1. Open `photo/index.html` in a modern browser.
2. Allow camera access.
3. Choose a filter.
4. Click 'Take Photos'.
5. Save photos or restart.

## Notes

- Requires a secure context (https or local file in supported browsers).
- Best tested in Chrome, Edge, or Firefox.
